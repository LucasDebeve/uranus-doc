# 🚀 Bienvenue dans le monde du Web !

Tu as toujours voulu savoir comment sont créés les sites Internet ? Ici, tu vas apprendre **à coder ton propre site Web** en partant de zéro.

Au programme : **HTML**, **CSS**, **JavaScript**, des jeux pour apprendre, et surtout un **super projet final** : créer ton propre site perso ou une mini page fan d’un sujet que tu aimes !

---

## 🧭 Objectifs du parcours

- Comprendre ce qu'est un site web
- Apprendre à structurer une page avec HTML
- Styliser avec CSS
- Ajouter de l'interactivité avec JavaScript
- Réaliser un projet personnel

---

## 📆 Déroulement de la formation

Chaque séance dure 1h30 (ou à adapter selon ton emploi du temps), avec un mix de théorie, de défis, et de création.

| Semaine | Thème | Objectif |
|--------|-------|----------|
| 1 | 🌍 Le Web, comment ça marche ? | Comprendre les bases (navigateur, serveur, URL…) |
| 2 | 🧱 HTML - La structure | Créer sa première page web |
| 3 | 🎨 CSS - La mise en forme | Styliser ton site comme un pro |
| 4 | 🧠 JavaScript - L'interactivité | Faire bouger les choses ! |
| 5 | 🛠 Projet : Ma page fan | Application de tout ce que tu as appris |
| 6 | 🎉 Présentation finale | Chaque élève présente son site |

---

## 💡 À la fin, tu sauras... 
- Développer une page Web
- Comprendre ce qui se cache derrière derrières tous les sites que tu utilises
- Présenter un projet technique à l'oral *(et avec fierté)*
