## Méthode

En base 2, on peut effectuer les même opérations qu'en base 10 et de la même manière. On peut s'appuyer sur les tables d'addition et de multiplication.

| $+$ | 0 | 1 |
| - | - | - |
| 0 | 0 | 1 |
| 1 | 1 | 10 |

| $\times$ | 0 | 1 |
| - | - | - |
| 0 | 0 | 0 |
| 1 | 0 | 1 |

> Exemple : $11_{2} + 10_{2}$

$$
\begin{aligned}
11_{2}& \\
+ \space \space 10_{2}& \\
\hline
101_{2}& \\
\end{aligned}
$$
